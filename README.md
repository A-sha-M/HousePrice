# House Price Prediction

## Project Overview
This project focuses on predicting house prices using Machine Learning techniques. The model is trained on housing data and analyzes various property features to estimate house prices accurately.

## Dataset
The project uses the `Housing.csv` dataset, which contains information about houses and their corresponding prices.

## Objectives
- Analyze housing data.
- Perform data preprocessing and feature analysis.
- Train a machine learning model for house price prediction.
- Evaluate model performance using appropriate metrics.
- Visualize relationships and prediction results.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Data Analysis
The project includes:
- Data cleaning and preprocessing.
- Exploratory Data Analysis (EDA).
- Correlation analysis between features.
- Visualization of house price distribution.
- Model training and testing.

## Visualizations
### Correlation Heatmap
Shows relationships between different housing features.

### Distribution of House Prices
Displays the distribution of house prices in the dataset.

### Actual vs Predicted House Prices
Compares actual house prices with model predictions to evaluate performance.

## Model Evaluation
The model's performance is assessed by comparing predicted values with actual house prices and analyzing prediction accuracy.

## Results
The machine learning model successfully learns patterns from housing data and provides house price predictions based on property characteristics.
